Black the sidebar and links

Sat, Apr 4, 2020
  • Today is a special day, April 4, 2020, Staturday. I will remember today forever.
    • China held a national mourning Saturday for martyrs who died in the fight against the novel coronavirus disease (COVID-19) outbreak and compatriots died of the disease.
    • During the commemoration, national flags flied at half-mast across the country and in all Chinese embassies and consulates abroad, and public recreational activities suspended across the country.
    • At 10:00 am today, Chinese people nationwide observed three-minute silence to mourn for the deceased, while air raid sirens and horns of automobiles, trains and ships wailed in grief.
  • Thus, I blacked my homepage.

Black the sidebar and links of blog

  • open site\themes\hyde\static\css\hyde.css
  • add following codes in the end of hyde.css, which colors the sidebar and links within blog posts to be black.
/* Black */
.theme-base-0g .sidebar {
  background-color: #000000;
}
.theme-base-0g .content a,
.theme-base-0g .related-posts li a:hover {
  color: #000000;
}
  • then, one can revise site\config.toml as follows
[params]
  themeColor = "theme-base-0g" # Black
  • of cause, one can color them in others.

Adjust the width of sidebar of blog

  • open site\themes\hyde\static\css\hyde.css
  • find and adjust width. The default is 18rem here.
@media (min-width: 48em) {
  .sidebar {
    position: fixed;
    top: 0;
    left: 0;
    bottom: 0;
    width: 14rem;
    text-align: left;
  }
}
  • then find and adjust Container as follows.
@media (min-width: 48em) {
  .content {
    /*max-width: 38rem;
    margin-left: 20rem;*/
	max-width: 44rem;
	margin-left: 20rem;
    margin-right: 2rem;
  }
}

@media (min-width: 64em) {
  .content {
    /*margin-left: 22rem;*/
	margin-left: 18rem;
    margin-right: 4rem;
  }
}
